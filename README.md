# 📚 Histoire de France - Ressources Pédagogiques

Ce dépôt propose une **histoire de France adaptée** à tous les niveaux, du **CP à la Terminale**, ainsi que pour les **adultes** et les **historiens**. Il inclut des **textes**, **sources historiques**, **images**, et **liens vers des vidéos** pour enrichir l'apprentissage.

## 🎯 Public Cible et Contenu

### 🏫 **École Primaire (CP à CM2)**
- **Format** : Résumés simplifiés, illustrations, activités ludiques (coloriages, quiz).
- **Exemples de thèmes** :
  - La Préhistoire (CP)
  - Les Gaulois et les Romains (CE1-CE2)
  - Les rois de France (CM1-CM2)

### 🏛️ **Collège (6ème à 3ème)**
- **Format** : Cours structurés, frises chronologiques, cartes, exercices.
- **Exemples de thèmes** :
  - La Révolution française (4ème-5ème)
  - Les deux guerres mondiales (3ème)

### 🎓 **Lycée (Seconde à Terminale)**
- **Format** : Analyses thématiques, documents historiques, débats, dissertations.
- **Exemples de thèmes** :
  - La construction de l'État-nation (Seconde)
  - Les régimes totalitaires (Première)
  - La Ve République (Terminale)

### 👨‍👩‍👧‍👦 **Adultes**
- **Format** : Synthèses accessibles, contextes socio-économiques, comparaisons historiques.
- **Exemples de thèmes** :
  - L'histoire des institutions françaises
  - Les grands mouvements sociaux

### 🔍 **Historiens**
- **Format** : Sources primaires (archives, textes originaux), bibliographies, études critiques.
- **Exemples de ressources** :
  - Les chroniques de Froissart
  - Les discours de Robespierre
  - Les travaux de Marc Bloch ou Fernand Braudel

## 📂 Ressources Multimédias

### 🖼️ **Images**
- Cartes historiques (ex: la Gaule en 50 av. J.-C.)
- Peintures (ex: *Le Serment des Horaces* de David)
- Photographies (ex: Paris sous Haussmann)

### 📜 **Textes**
- Extraits de lois (ex: Edit de Nantes, Déclaration des Droits de l'Homme)
- Discours (ex: Appel du 18 Juin)
- Traités (ex: Traité de Versailles)

### 🎥 **Vidéos**
- Liens vers des documentaires (ex: *Les Ailes de la Nature* pour la Préhistoire)
- Conférences (ex: cours du Collège de France)
- Animations (ex: frises chronologiques interactives)

### 📚 **Sources**
- Archives numérisées (ex: Gallica, BnF)
- Articles scientifiques (ex: revues comme *Annales. Histoire, Sciences Sociales*)
- Bibliographies thématiques

## 🛠️ Comment Contribuer ?

1. **Fork** ce dépôt.
2. Crée une **branche** pour ta contribution (ex: `feature/ajout-revolution-francaise`).
3. Ajoute ton contenu dans le dossier approprié.
4. Ouvre une **Pull Request** avec une description claire.

### 📌 Règles à respecter :
- **Respecter les droits d'auteur** : Utilise des images/libres de droits ou sous licence CC.
- **Citer les sources** : Toujours indiquer l'origine des textes ou images.
- **Adapter le langage** : Utiliser un vocabulaire adapté au public cible.
